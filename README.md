# 🚀 CRUD com Spring Boot + H2 Database

<div align="center">

Projeto desenvolvido para demonstrar a criação de um CRUD completo utilizando **Spring Boot** e **H2 Database**.

</div>

---

## 📌 Tecnologias Utilizadas

- Java 23 (JDK 23)
- Spring Boot
- Spring Data JPA
- H2 Database
- Lombok
- Maven

---

## 📚 Funcionalidades

O projeto realiza operações básicas de um CRUD:

- ✅ Criar registros
- ✅ Listar registros
- ✅ Atualizar registros
- ✅ Remover registros

---

## 🧠 Conceitos Trabalhados

Durante o desenvolvimento deste projeto, são aplicados conceitos importantes como:

- Estruturação de projetos com Spring Boot
- Criação de APIs REST
- Integração com banco de dados H2
- Utilização do Spring Data JPA
- Organização em camadas
- Uso de dependências com Maven
